# 新能源铝电解电容应用方案

新能源领域对电容的寿命、可靠性和耐候性要求极为严苛。华裕电子产品广泛应用于光伏逆变器、风力发电变流器、储能系统（ESS）等新能源设备。

## 新能源设备的电容关键要求

| 要求 | 说明 |
|------|------|
| 超长寿命 | 光伏逆变器设计寿命 20~25 年，电容是短板元件 |
| 高低温特性 | 户外机柜 -30℃~+70℃ 环境温度 |
| 高纹波承受能力 | 大功率逆变器纹波电流大 |
| 高可靠性 | 电站场景维护困难，零失效是目标 |

## 光伏逆变器 PV Inverter

光伏逆变器是华裕电子核心应用领域之一，CD298、CD13HL 系列专为光伏设计。

| 功率段 | DC-Link 推荐系列 | 电压/容量 | 关键特性 |
|--------|-----------------|------------|----------|
| 微型（< 2 kW） | CD294 | 200~400V, 470~1,500μF | 紧凑, 经济 |
| 组串式（2~100 kW） | CD298 | 350~500V, 2,200~10,000μF | 10000h 超长寿命 |
| 集中式（> 100 kW） | CD13HL | 350~500V, 3,900~15,000μF | 超长寿命, 高可靠 |
| 户用储能逆变器 | CD13L | 200~450V, 470~4,700μF | 长寿命, 经济 |

## 风力发电 Wind Power

风电变流器功率大，对电容的纹波承受能力和寿命要求极高。

| 设备类型 | 推荐系列 | 说明 |
|----------|----------|------|
| 双馈式变流器 | CD13HL, CD298 | 超长寿命，高纹波 |
| 全功率变流器 | CD13L, CD92L | 低压侧大容量 |
| 变桨系统后备电源 | CD98SK | 超级电容，瞬间功率补偿 |

## 储能系统 Energy Storage System (ESS)

储能 PCS（功率转换系统）对 DC-Link 电容要求与光伏类似，但充放电频次更高。

| 应用场景 | 推荐系列 | 特点 |
|----------|----------|------|
| 电网侧储能 | CD13HL | 超长寿命，高可靠 |
| 工商业储能 | CD13L, CD298 | 长寿命，性价比 |
| 户用储能 | CD294, CD13L | 紧凑，经济 |
| 储能变流器 DC-Link | CD13HL, CD92L | 高纹波，长寿命 |

## 选型要点

1. **寿命优先：** 新能源设备设计寿命长，优先选择 CD298（10000h）或 CD13HL（10000h）系列。
2. **温度降额：** 户外机柜夏季内部温度可达 70℃ 以上，建议选择 105℃ 系列并充分降额使用。
3. **纹波电流：** 大功率逆变器纹波电流大，建议选择低 ESR 系列（CD298、CD13HL）。
4. **电压选择：** 光伏直流侧电压波动大，建议选择额定电压 ≥ 最大直流母线电压 ÷ 0.8。
