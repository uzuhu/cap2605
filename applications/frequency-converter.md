# 变频器铝电解电容选型方案

变频器是工业自动化的核心设备，其内部直流母线电容（DC-Link）对系统稳定性和寿命起着决定性作用。华裕电子针对各类变频器提供专业的电容选型方案。

## 变频器对电容的关键要求

变频器工作时，整流后的直流母线电压存在纹波，逆变桥的高频开关动作也会在母线上产生高频纹波电流。因此，变频器用电容需满足以下要求：

| 要求 | 说明 |
|------|------|
| 高纹波电流承受能力 | 变频器工作电流大，纹波电流有效值高 |
| 长寿命 | 工业设备通常要求10年以上使用寿命 |
| 高可靠性 | 工业现场环境复杂，电容失效会导致设备停机 |
| 低ESR | 降低纹波电压，减少电容发热 |

## 选型计算参考

变频器直流母线电容的选择可参考以下经验公式：

<div class="formula-block">
  <code>C ≥ (I_ripple)² / (ΔV × f_sw × V_dc)</code>
</div>

| 变量 | 含义 |
|------|------|
| C | 所需电容量（F） |
| I_ripple | 纹波电流有效值（A） |
| ΔV | 允许纹波电压（V） |
| f_sw | 开关频率（Hz） |
| V_dc | 直流母线电压（V） |

**选型要点：** 中大功率变频器建议选用 CD13NH（105℃）或 CD13HL（10000h）系列，工作电压不超过额定值的 80%。
