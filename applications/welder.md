# 电焊机电容方案

电焊机（Welding Machine）是铝电解电容的重要应用场景之一。无论是 MIG/MAG 气体保护焊、TIG 钨极氩弧焊，还是等离子切割设备，都需要大容量电容提供稳定的焊接电流和抑制电压波动。华裕电子针对电焊机应用提供专业的电容选型方案。

## 电焊机对电容的核心要求

| 要求 | 说明 |
|------|------|
| **大容量** | 电焊机需要电容提供瞬时大电流，容量需足够大 |
| **高纹波电流** | 焊接电流大，电容需承受高纹波 |
| **低 ESR** | 降低发热，提高焊接稳定性 |
| **耐高电压** | 等离子切割等应用电压较高 |
| **抗振动** | 便携式电焊机现场使用，需耐振动 |

## 电焊机电容配置

### 1. 输入整流滤波电容

连接在电焊机输入端，将市电整流后的脉动直流滤成平稳直流，为后续逆变器提供稳定电源。

**推荐系列：CD135 / CD136（螺栓型，大容量）**

| 电焊机功率 | 输入电压 | 推荐电容 |
|-----------|---------|---------|
| ≤ 3kW | 单相 220V | CD135 400V 2200μF |
| 3~10kW | 单相 220V / 三相 380V | CD136 400V 4700μF × 2 |
| 10~20kW | 三相 380V | CD13L 400V 10000μF × 2 |
| ≥ 20kW | 三相 380V | CD13NH 400V 27000μF × 多颗 |

### 2. 直流母线支撑电容（DC-Link）

连接在逆变桥前端，稳定直流母线电压，吸收开关纹波。

**推荐系列：CD294 / CD297 / CD13L**

### 3. 输出滤波电容

连接在电焊机输出端，滤除高频开关噪声，使焊接电弧更稳定。

## 电焊机电容选型要点

### 1. 容量计算

电焊机输入滤波电容容量可参考：

$$C = \frac{P_{rated}}{2\pi f V_{dc} \Delta V}$$

其中：
- $P_{rated}$：电焊机额定功率（W）
- $f$：市电频率（50/60Hz）
- $V_{dc}$：整流后直流电压（V）
- $\Delta V$：允许的电压纹波（V）

### 2. 电压选择

| 输入电源 | 整流后直流电压 | 推荐电容耐压 |
|---------|--------------|------------|
| 单相 220V | ~310V | 400V |
| 三相 380V | ~537V | 500V / 550V |

### 3. 纹波电流校核

电焊机工作时电流波动大，电容纹波电流应按最大焊接电流的 1.5~2 倍校核。

## 使用注意事项

1. **安装方向**：螺栓型电容安装时应确保端子受力均匀，避免倾斜导致密封不良
2. **散热设计**：大电流电焊机电容发热大，需确保良好散热
3. **定期维护**：每 1~2 年检测一次电容容量和 ESR
4. **防爆间距**：电容上方需留足够空间，确保防爆阀动作时不受阻
