# 逆变器电容选型

逆变器 DC-Link 母线电容是关键储能器件，需承受高压纹波与长期稳定工作。根据功率和应用场景推荐以下系列：

| 类型 | 推荐系列 | 说明 |
|------|----------|------|
| 光伏组串式逆变器 | CD297, CD298 | 350~500V, 2,200~10,000μF |
| 光伏集中式逆变器 | CD13HL | 200~500V, 3,900~15,000μF |
| 储能逆变器 | CD13L, CD92L | 宽电压, 低阻抗 |
| 通用工业逆变器 | CD294, CD13N | 标准品, 经济型 |

**选型要点：** 逆变器母线电容工作电压建议取额定电压的 70%~80%，并关注纹波电流有效值是否在许可范围内。
