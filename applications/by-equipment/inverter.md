# 逆变器电容方案

逆变器（Inverter）是光伏发电、不间断电源（UPS）、电动汽车驱动等系统的核心部件，负责将直流电（DC）转换为交流电（AC）。铝电解电容在逆变器中承担着直流支撑（DC-Link）、输出滤波、缓冲吸收等关键功能，其性能直接影响逆变器的效率、可靠性和使用寿命。

## 逆变器对电容的核心要求

| 要求 | 说明 |
|------|------|
| **高纹波电流能力** | 逆变器开关频率高（通常 2~20kHz），电容需承受大纹波电流 |
| **低 ESR** | 低等效串联电阻可减少发热，提高系统效率 |
| **高额定电压** | DC-Link 电容需承受母线电压及电压尖峰 |
| **长寿命** | 工业及光伏逆变器设计寿命 10~20 年，电容寿命需匹配 |
| **耐高温** | 密闭机箱内温度高，105℃ 产品更适合 |
| **高可靠性** | 7×24 小时连续运行，故障代价极高 |

## 逆变器电容配置方案

典型逆变器电路包含以下电容配置：

### 1. 直流支撑电容（DC-Link Capacitor）

连接在直流母线正负极之间，作用是：
- 吸收开关动作产生的高频纹波电流
- 稳定直流母线电压，抑制电压波动
- 提供瞬时功率补偿

**推荐系列：CD294 / CD297 / CD298**

| 参数 | 典型值 |
|------|--------|
| 额定电压 | 450V、500V、550V |
| 容量范围 | 470μF ~ 3300μF |
| 纹波电流 | 可达 5.0A @ 120Hz, 105℃ |
| 预期寿命 | CD294: 5000h / CD298: 10000h @ 105℃ |
| 端子类型 |  snap-in（牛角型） |

### 2. 输出滤波电容（Output Filter Capacitor）

连接在逆变器输出端，用于滤除 PWM 谐波，改善输出波形质量。

**推荐系列：CD295 / CD17FX**

| 参数 | 典型值 |
|------|--------|
| 额定电压 | 350V ~ 500V |
| ESR | CD17FX 系列极低 ESR，适合高频应用 |
| 适用场景 | 高频开关逆变器、高性能并网逆变器 |

### 3. 缓冲吸收电容（Snubber Capacitor）

并联在功率器件（IGBT/SiC MOSFET）两端，吸收电压尖峰，保护功率器件。

> **提示：** 缓冲电路通常使用薄膜电容，但在小功率逆变器中也可选用 ESR 极低的铝电解电容（CD17FX）。

## 光伏逆变器专用推荐

| 功率等级 | 推荐 series | 说明 |
|----------|-------------|------|
| 小功率（< 10kW）| CD294 | 性价比高，寿命 5000h |
| 中功率（10~50kW）| CD297 | 更长的 8000h 寿命 |
| 大功率（> 50kW）| CD298 | 10000h 长寿命，高可靠性 |
| 高频高效机型 | CD17FX | 极低 ESR，适合 SiC/GaN 高频开关 |

## 选型要点

1. **电压裕量**：DC-Link 电容额定电压应 ≥ 最大直流母线电压 × 1.1，建议选用 450V 或 500V 规格
2. **容量计算**：DC-Link 电容容量应满足 `C ≥ I_ripple / (2πf × ΔV)`，其中 ΔV 为允许的电压纹波
3. **纹波电流校核**：电容的额定纹波电流应大于实际纹波电流，必要时采用多颗并联
4. **温度降额**：若实际工作温度高于 85℃，应选用 105℃ 规格（CD13NH、CD298）
5. **寿命匹配**：光伏逆变器设计寿命 20 年以上，建议选用 CD298（10000h @ 105℃）
