# 工业电源铝电解电容选型方案

工业电源是铝电解电容最核心的应用领域之一。开关电源（SMPS）、DC-DC 转换器、PFC 电路等工业电源设备，对电容的可靠性、纹波承受能力和使用寿命都有较高要求。

## 工业电源典型电路中的电容作用

| 电路位置 | 电容作用 | 关键要求 |
|----------|----------|----------|
| 输入滤波级 | 滤除电网侧纹波 | 大容量，高纹波承受能力 |
| PFC 输出级 | 稳定直流母线电压 | 低 ESR，耐高温 |
| DC-DC 输出端 | 输出滤波，稳定电压 | 低 ESR，长寿命 |
| 辅助电源 | 控制线路供电 | 小体积，高可靠 |

## 推荐产品系列

| 应用场景 | 推荐系列 | 关键特性 | 典型电压/容量 |
|----------|----------|----------|----------------|
| 通用开关电源输入滤波 | CD92 / CD135 | 低压大容量 | 25~100V, 1,000~100,000μF |
| 开关电源 DC-Link | CD294 / CD297 | 长寿命, 中高压 | 350~500V, 470~3,300μF |
| PFC 输出滤波 | CD13L | 长寿命, 低 ESR | 400~450V, 680~2,700μF |
| 高频开关电源 | CD17FX | 极低 ESR/ESL | 200~400V, 100~1,000μF |
| 高温环境电源 | CD13NH | 105℃ 高温 | 160~500V, 270~39,000μF |
| 通信电源 | CD293 / CD294 | 低阻抗，长寿命 | 16~100V / 160~500V |

## 选型建议

1. **电压选择：** 工作电压建议为额定电压的 70%~80%，可显著延长寿命。
2. **温度考虑：** 电源内部温升高，优先选择 105℃ 系列（CD294/CD13L/CD13HL）。
3. **寿命要求：** 长寿命需求选择 CD294（5000h）、CD298（10000h）或对应螺栓型。
4. **纹波电流：** 高频大纹波场景选 CD295（高纹波）或 CD17FX（高频低阻）。
5. **安规认证：** 出口产品需确认电容符合 EN 62368、UL 等安规要求。
