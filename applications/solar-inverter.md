# 光伏逆变器铝电解电容选型方案

光伏逆变器（Solar Inverter）将太阳能电池板产生的直流电转换为可并网的交流电，是光伏发电系统的核心设备。铝电解电容在光伏逆变器中承担着直流支撑（DC-Link）、MPPT 滤波、输出滤波等关键功能，其寿命和可靠性直接决定光伏电站的发电效率和运维成本。

## 光伏逆变器对电容的核心要求

| 要求 | 说明 |
|------|------|
| **超长寿命** | 光伏电站设计寿命 20~25 年，电容寿命需尽量匹配 |
| **高纹波电流** | 并网逆变器满载时纹波电流大，电容需承受 |
| **耐高低温** | 户外昼夜温差大，电容需在 -40℃ ~ +85℃ 稳定工作 |
| **高额定电压** | 直流侧电压可达 500V+，电容需足够耐压 |
| **高可靠性** | 户外无人值守，故障更换成本高 |

## 光伏逆变器电容配置

### 1. 直流侧支撑电容（DC-Link）

连接在光伏阵列输入与逆变器桥臂之间，作用是稳定直流母线电压，吸收 PWM 谐波电流。

**光伏逆变器专用推荐系列：CD297 / CD298**

| 参数 | 典型值 |
|------|--------|
| 额定电压 | 450V、500V、550V |
| 容量范围 | 820μF ~ 4700μF（依功率等级） |
| 纹波电流 | 4.0A ~ 7.0A @ 120Hz, 105℃ |
| 预期寿命 | CD297: 8000h / CD298: 10000h @ 105℃ |
| 温度范围 | -40℃ ~ +105℃ |

### 2. MPPT 电路滤波电容

最大功率点跟踪（MPPT）电路需要电容滤除高频开关噪声，确保最大功率追踪精度。

**推荐系列：CD294 / CD295**

### 3. 交流输出滤波电容

连接在逆变器输出端，滤除 PWM 谐波，确保并网电流质量符合 IEEE 1547 / IEC 61727 标准。

## 光伏应用选型要点

### 1. 寿命计算

光伏逆变器设计寿命通常 20~25 年，电容寿命计算：

$$L_{actual} = L_{rated} \times 2^{\frac{T_{rated} - T_{actual}}{10}} \times (\frac{V_{rated}}{V_{actual}})^n$$

> 实际工程中，建议选用 **CD298（10000h @ 105℃）** 系列，在 60℃ 实际工作温度下寿命可达约 20 年。

### 2. 电压选择

- 单相并网：DC-Link 电压约 400V，建议选用 450V 规格
- 三相并网：DC-Link 电压约 700~800V，建议选用 500V × 2 串联或 550V 单颗
- 考虑电压尖峰，额定电压应留 15% 以上裕量

### 3. 容量选择

DC-Link 电容容量应满足电压纹波要求：

$$\Delta V = \frac{I_{load}}{2\pi f_{sw} C}$$

一般要求 $\Delta V / V_{dc} ≤ 5\%$，据此计算最小容量。
