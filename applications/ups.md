# UPS不间断电源铝电解电容选型方案

UPS（Uninterruptible Power Supply，不间断电源）在市电异常时为负载提供持续、稳定的电力供应，广泛应用于数据中心、医疗设备、工业控制、通信基站等关键场景。铝电解电容在 UPS 中承担着整流滤波、直流支撑、逆变器输出滤波等重要功能，其可靠性直接关系到 UPS 的供电连续性。

## UPS 对电容的核心要求

| 要求 | 说明 |
|------|------|
| **高可靠性** | UPS 为关键负载供电，电容故障可能导致供电中断 |
| **长寿命** | 数据中心 UPS 7×24 小时运行，设计寿命 10~15 年 |
| **高纹波电流** | 双变换在线式 UPS 纹波电流大 |
| **耐高低温** | 数据中心机房温度可控，但户外型 UPS 环境多样 |
| **大容量** | 整流输出端需要大容量电容滤除纹波 |

## UPS 系统电容配置

典型双变换在线式 UPS 电路包含以下电容：

### 1. 整流器输出滤波电容

连接在整流器输出端，将市电整流后的脉动直流滤成平稳直流。

**推荐系列：CD135 / CD136（螺栓型，大容量）**

| 参数 | 典型值 |
|------|--------|
| 额定电压 | 80V ~ 400V（依 UPS 输入电压） |
| 容量范围 | 1000μF ~ 100,000μF（依功率等级） |
| 纹波电流 | 可达 10A+ @ 120Hz |
| 端子类型 | 螺栓型（M5/M6 螺纹） |

### 2. 直流母线支撑电容（DC-Link）

连接在有源整流器与逆变器之间，稳定直流母线电压。

**推荐系列：CD294 / CD297 / CD13L**

### 3. 逆变器输出滤波电容

连接在 UPS 逆变器输出端，滤除 PWM 谐波，提供纯净正弦波。

**推荐系列：CD295 / CD17FX**

### 4. 电池端滤波电容

连接在电池正负极之间，滤除电池侧纹波，延长电池寿命。

**推荐系列：CD92 / CD135（低压大容量）**

## UPS 电容选型要点

### 1. 在线式 vs 后备式

| UPS 类型 | 电容要求 | 推荐系列 |
|---------|---------|---------|
| 后备式（Off-line）| 要求较低，成本敏感 | CD92、CD292 |
| 在线互动式（Line-interactive）| 中等要求 | CD294、CD135 |
| 双变换在线式（Double-conversion）| 高可靠性、长寿命 | CD297、CD298、CD13L |

### 2. 电池电压匹配

UPS 电池电压通常为 12V / 24V / 36V / 48V / 96V / 192V 等，电容耐压需匹配：

| 电池电压 | 推荐电容耐压 |
|---------|------------|
| 12V ~ 48V | 63V / 80V |
| 96V ~ 192V | 200V / 250V |
| 192V+ | 350V / 400V |

### 3. 寿命要求

| 应用场景 | 设计寿命 | 推荐系列 |
|---------|---------|---------|
| 家用/办公室 | 3~5 年 | CD92、CD292 |
| 商用/工业 | 8~10 年 | CD294、CD135 |
| 数据中心 | 10~15 年 | CD297、CD298、CD13L |

## 维护保养建议

1. **定期检测**：每年检测一次电容容量和 ESR，及时发现老化电容
2. **环境控制**：保持 UPS 机房温度 20~25℃，可显著延长电容寿命
3. **防尘防潮**：定期清洁 UPS 内部，防止粉尘和潮湿导致电容腐蚀
4. **更换标准**：电容容量低于标称值 80% 或 ESR 超过标称值 2 倍时建议更换
