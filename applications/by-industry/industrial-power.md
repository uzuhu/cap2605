# 工业电源电容方案

工业电源是铝电解电容最核心的应用领域之一。针对不同功率和拓扑结构的工业电源，华裕电子提供以下电容方案。

## 推荐产品系列

| 应用场景 | 推荐系列 | 关键特性 | 典型电压/容量 |
|----------|----------|----------|---------------|
| 通用开关电源输入滤波 | CD92 / CD135 | 低压大容量 | 25~100V, 1,000~100,000μF |
| 开关电源DC-Link | CD294 / CD297 | 长寿命, 中高压 | 350~500V, 470~3,300μF |
| PFC输出滤波 | CD13L | 长寿命, 低ESR | 400~450V, 680~2,700μF |
| 低压大电流电源 | CD136 | 高纹波, 长寿命 | 16~100V, 6,800~150,000μF |
| 高频开关电源 | CD17FX | 极低ESR/ESL | 200~400V, 100~1,000μF |
| 高温环境电源 | CD13NH | 105℃高温 | 160~500V, 270~39,000μF |
| 超级储能 | CD98SK | 超级电容 | 16~25V, 0.5~2F |

## 选型建议

1. **电压选择**：工作电压建议为额定电压的70%~80%
2. **温度考虑**：高温环境下优先选择CD13NH(105℃)系列
3. **寿命要求**：长寿命需求选择CD294(5000h)、CD298(10000h)
4. **纹波电流**：高频大纹波场景选CD295或CD17FX
