# 伺服驱动器铝电解电容选型方案

伺服驱动器（Servo Drive）是工业自动化中的核心部件，用于精确控制伺服电机的位置、速度和转矩。铝电解电容在伺服驱动器中主要承担直流母线支撑、再生制动能量吸收、输出滤波等功能，对驱动器的动态响应和可靠性至关重要。

## 伺服驱动器对电容的核心要求

| 要求 | 说明 |
|------|------|
| **高纹波电流** | 伺服电机频繁加减速，母线纹波电流大 |
| **低 ESR** | 降低发热，提高动态响应能力 |
| **耐反向电压尖峰** | 再生制动时母线电压升高，电容需耐受 |
| **长寿命** | 工业现场 7×24 小时运行，寿命要求高 |
| **高可靠性** | 自动化产线停机代价高，电容故障率需极低 |

## 伺服驱动器电容配置

### 1. 直流母线支撑电容（DC-Link）

连接在整流器输出与逆变器输入之间，作用是稳定直流母线电压，吸收开关纹波。

**推荐系列：CD294 / CD297 / CD13L**

| 参数 | 典型值 |
|------|--------|
| 额定电压 | 350V、400V、450V |
| 容量范围 | 470μF ~ 2200μF（依功率等级） |
| 纹波电流 | ≥ 3.0A @ 120Hz |
| 预期寿命 | 5000h ~ 10000h @ 105℃ |

### 2. 再生制动吸收电容

伺服电机减速时，机械能回馈至直流母线，导致电压升高。制动电路（Brake Chopper）配合电容吸收这部分能量。

| 功率等级 | 推荐 series | 容量参考 |
|----------|-------------|---------|
| ≤ 1kW | CD92 | 470 ~ 1000μF / 400V |
| 1 ~ 5kW | CD294 | 1000 ~ 2200μF / 400V |
| 5 ~ 15kW | CD297 | 2200 ~ 4700μF / 400V |
| ≥ 15kW | CD298（多颗并联）| 依计算确定 |

### 3. 控制单元电源滤波电容

为驱动器内部控制电路（DSP、FPGA、传感器）提供稳定电源。

**推荐系列：CD292 / CD91A**（小体积，低压）

## 选型计算参考

### 直流母线电容容量估算

$$C_{min} = \frac{I_{peak} \times \Delta t}{\Delta V_{max}}$$

其中：
- $I_{peak}$：峰值电流（A）
- $\Delta t$：最大负载突变时间（s）
- $\Delta V_{max}$：允许的最大母线电压波动（V）

### 纹波电流校核

伺服驱动器输出电流含高频 PWM 分量，电容纹波电流近似：

$$I_{ripple} \approx I_{rms} \times \sqrt{D(1-D)}$$

其中 D 为 PWM 占空比。

## 选型要点

1. **电压裕量**：母线电容额定电压 ≥ 最大直流母线电压 × 1.15
2. **温度等级**：伺服驱动器机箱内部温度高，优先选用 105℃ 规格
3. ** ESR 要求**：低 ESR 有助于降低温升，CD17FX 系列最适合高频应用
4. **安装方式**：大容量建议选用 snap-in（牛角型）端子，确保连接可靠
5. **并联设计**：大功率驱动器采用多颗电容并联，降低 ESR 并提高纹波能力
