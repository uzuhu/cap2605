# 型号命名规则 Part Number System

## 命名格式

## 容量代码 Capacitance Code (μF)

容量代码为3位数字，前2位为容量有效数字，第3位为10的幂次。例如：
- **101** = 10 × 10¹ = 100μF
- **102** = 10 × 10² = 1,000μF
- **222** = 22 × 10² = 2,200μF
- **682** = 68 × 10² = 6,800μF
- **103** = 10 × 10³ = 10,000μF
- **474** = 47 × 10⁴ = 470,000μF

## 容量偏差 Capacitance Tolerance

| 符号 | 偏差范围 |
|------|----------|
| M | ±20% |
| L | ±15% |
| K | ±10% |
| R | -0+20% |
| F | -0+30% |
| V | -10+20% |
| Q | -10+30% |
| T | -10+50% |
| U | -10+75% |

## 尺寸代码 Diameter Code

尺寸代码为6位数字，前2位为直径(mm)，中间2位为高度(mm)。例如：
- **022030** = Φ22×30mm
- **051080** = Φ51×80mm
- **077155** = Φ77×155mm

## 特殊代码 Special Code

| 符号 | 含义 |
|------|------|
| S | 带安装螺栓 With Stud |
| B | 带固定支架 With Bracket |

## 命名示例 Examples

**例1:** `LE 1H 103 M 030050`
- LE = CD294系列
- 1H = 50V
- 103 = 10,000μF
- M = ±20%
- 030050 = Φ30×50mm
→ **CD294 50V 10,000μF ±20% Φ30×50mm**

**例2:** `BP 2G 682 M 077155 S`
- BP = CD135系列
- 2G = 400V
- 682 = 6,800μF
- M = ±20%
- 077155 = Φ77×155mm
- S = 带安装螺栓
→ **CD135 400V 6,800μF ±20% Φ77×155mm 带安装螺栓**
