# 螺栓型电解电容 Screw Terminal Type

螺栓型(Screw Terminal)铝电解电容采用螺栓引出端子，适用于大电流、高功率应用。端子通过螺栓与铜排或电缆连接，确保低接触电阻和高可靠性。
